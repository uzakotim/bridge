# 02-mvp  
  
  
# MVP Specification  
  
## MVP Features  
  
## 1. User Profile  
  
Fields:  
  
- name  
- current skills  
- years experience  
- target job  
- location  
- preferred work type  
  
Example:  
  
Target:  
Junior Backend Developer  
  
Skills:  
Python  
SQL  
Git  
  
---  
  
## 2. Job Description Analyzer  
  
Input:  
  
Job description text  
  
Output:  
  
Required skills:  
  
Technical:  
- Python  
- PostgreSQL  
- Docker  
  
Soft:  
- communication  
- teamwork  
  
  
Experience:  
- REST APIs  
- cloud deployment  
  
  
Store extracted data.  
  
---  
  
## 3. Skill Gap Analysis  
  
Compare:  
  
User skills  
vs  
Market requirements  
  
  
Output:  
  
Strong:  
Python  
  
Missing:  
Docker  
AWS  
Testing  
  
  
Priority score:  
  
High:  
Docker  
  
Medium:  
AWS  
  
Low:  
Kubernetes  
  
  
---  
  
## 4. Small Project Generator  
  
Generate:  
  
Project idea 1:  
- short explanation  
- links to resources  
- project idea  


Project idea 2:  
- short explanation  
- links to resources  
- project idea  

Each item:  
- explanation  
- resources  
- project idea  
  
  
---  
  
## 5. Job Matching  
  
User profile:  
  
Find jobs where:  
  
skills match > 70%  
  
Show:  
  
- match score  
- missing skills  
- recommendations

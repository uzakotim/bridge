# AI Career Agent Product Vision  
  
## Goal  
  
Build an AI-powered career assistant that helps users become qualified for a target profession.  
  
Example:  
User wants:  
"Junior Robotics Software Engineer"  
  
The system analyzes:  
- job descriptions  
- required skills  
- technologies  
- common responsibilities  
  
Then creates:  
- skill gap analysis  
- learning roadmap  
- project recommendations  
- job matching  
  
## Core User Flow  
  
1. User creates profile  
2. User selects target role  
3. User uploads CV or enters skills  
4. System analyzes market requirements  
5. System creates personalized roadmap  
6. User tracks progress  
7. System recommends jobs  
  
## Initial Target Market  
  
Start narrow:  
  
Junior software engineers  
  
Later expand:  
- robotics  
- AI engineers  
- data scientists  
- frontend developers  
- cybersecurity  
  
## Main Value Proposition  
  
"Know exactly what skills you need and what to build to get hired."  
  
## Do NOT build initially  
  
Avoid:  
- automatic job applications  
- full AI tutor  
- all professions  
- complex social network  
  
First prove:  
Can we help someone close a skill gap?
